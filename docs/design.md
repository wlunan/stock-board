# 设计文档

## 1. 项目概述

### 1.1 目标
开发一个轻量级Windows桌面股票看板，用于"摸鱼"时查看股票行情。

### 1.2 核心需求
- 轻量：单文件 < 10MB，启动快
- 低调：小窗口、透明、可隐藏
- 实时：定时刷新行情数据
- 多品种：股票、ETF、黄金

## 2. 功能设计

### 2.1 数据展示
```
┌─────────────────────────────┐
│ 📈 股票摸鱼看板              │
├─────────────────────────────┤
│ 上证指数   3,250.12  +0.52% │
│ 深证成指  10,850.30  -0.18% │
│ 沪深300ETF  4.125   +0.35% │
│ 黄金ETF    4.850   +1.20% │
├─────────────────────────────┤
│ 更新: 14:32:15              │
└─────────────────────────────┘
```

### 2.2 涨跌标识
- 上涨：绿色（+X.XX%）
- 下跌：红色（-X.XX%）
- 平盘：灰色（0.00%）

### 2.3 窗口特性
- 置顶显示
- 透明度可调（50%-100%）
- 可拖拽移动
- 无边框设计
- 最小化到托盘

### 2.4 快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+H | 隐藏/显示 |
| Ctrl+Shift+Q | 退出 |
| Ctrl+滚轮 | 调整透明度 |

## 3. 技术架构

### 3.1 整体架构
```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  UI Layer   │◄───│  Data Layer │◄───│  API Layer  │
│  (Fyne GUI) │    │  (Stock)    │    │  (HTTP)     │

```

### 3.2 模块划分

#### main.go
- 程序入口
- 初始化配置
- 启动GUI
- 注册快捷键

#### config.go
- 配置文件读写
- 默认配置
- 配置验证

#### stock.go
- 股票数据结构
- 数据获取（HTTP请求）
- 数据解析（JSON/文本）
- 缓存管理

#### ui.go
- 主窗口创建
- 股票列表组件
- 数据刷新显示
- 窗口拖拽
- 系统托盘

#### hotkey.go
- 全局快捷键注册
- 快捷键事件处理

## 4. 数据源设计

### 4.1 API选择
使用新浪财经API（免费、稳定）：

```
# 股票/指数
https://hq.sinajs.cn/list=sh000001,sz399001

# ETF
https://hq.sinajs.cn/list=sh510300,sh518880

# 黄金（上海金）
https://hq.sinajs.cn/list=hf_GC
```

### 4.2 数据格式
新浪返回格式：
```
var hq_str_sh000001="上证指数,3250.12,...";
```

解析为：
```go
type StockData struct {
    Code      string    // 代码
    Name      string    // 名称
    Price     float64   // 当前价
    Change    float64   // 涨跌额
    ChangePct float64   // 涨跌幅
    Time      string    // 更新时间
}
```

### 4.3 备用数据源
- 东方财富API
- 腾讯股票API
